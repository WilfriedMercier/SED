A general named column to assicate to an Enum object.

__init__(name: str, unit: astropy.units.core.Unit, end: str = '', log=False) → None[source]¶

Code author: Wilfried Mercier - IRAP <wilfried.mercier@irap.omp.eu>

Init named column.

Parameters

  • name (str) – name of the column when included in a Table

  • unit (Unit) – unit of the column

  • end (str) – (Optional) end string to append to the associate enum name

  • log (bool) – (Optional) whether this column holds log values. This will trigger a conversion to the power of 10 if True.

Raises

TypeError –

  • if name is not of type str

  • if unit is not of type Astropy Unit

  • if end is not of type str
